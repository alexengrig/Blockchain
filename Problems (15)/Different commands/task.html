Different commands Open on Hyperskill

Write two classes which represent the Command pattern, CommandMove and CommandPickItem.

They should implement the entity movement on the 2D and item picking. The x-axis on the map goes left to right, the y-axis stretches bottom to top.

The inventory of the entity has getInventoryLength() slots starting from 0, where each slot can contain only one item. The entity should put an item into a slot with least possible index (slots with nothing inside is null). If all slots of an entity are filled, then the entity should not pick this item for its inventory.



Memory limit: 256 Mb
Time limit: 5s
